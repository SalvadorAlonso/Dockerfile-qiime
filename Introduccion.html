    LA IMAGEN DOCKER

    La imagen Docker contiene una serie de comandos de instalación y actualización de librerías de dos programas, éstos son Ipython y QIIME. QIIME es el programa para el que se diseñó la imagen Docker pero para trabajar con él necesitamos instalarnos un intérprete que haga de intermediario entre nosotros y QIIME.

    El intérprete es IPython, el cual se ejecuta a través de la Terminal y tiene un diseño de lectura basado en el lenguaje Python.

    Una de las ventajas de IPython es el empleo del comando 'ipython notebook', este comando abre una ventana de un navegador web que nos permite trabajar en un entorno interactivo en el que se combina la ejecución de códigos, operaciones matemáticas y obtención de imágenes como resultado de las operaciones mencionadas.

    QIIME

    Una vez conocemos cómo funciona de manera somera Docker y la manipulación de sus imágenes y contenedores vamos a trabajar con lo que hemos instalado: QIIME e IPython.

    QIIME es un pipeline que realiza análisis de cadenas de ADN microbiano, opera sobre la cadena de ADN 'en bruto' y genera tablas e imágenes que reflejan los resultados obtenidos.

    Aquí un pequeño tutorial sobre el uso de QIIME para principiantes

    Para una mayor información sobre QIIME se recomienda realizar los diferentes tutoriales que aparecen en su página web: TUTORIALES DE QIIME

    Los tutoriales más básicos con los que se recomienda empezar son QIIME Illumina Overview Tutorial (QIIME illumina Overview Tutorial) y the QIIME 454 Overview Tutorial (QIIME Overview Tutorial).

    IPYTHON E IPYTHON NOTEBOOK

    Para poder trabajar con QIIME es necesario usar un intérprete, el que incorpora QIIME por defecto es el intérprete IPython (http://ipython.org/). IPython presenta un sistema de entrada de datos basado en el lenguaje Python y una librería (IPython notebook) que abre una ventana de un navegador web a través de la cual podemos visualizar las gráficas o estadísticas que se generan las operaciones que se han ido realizando.
